# Usage Guide

This reference document provides additional context that Claude loads only when needed (progressive disclosure level 3).

## When to Reference This File

- When the user asks for detailed usage patterns
- When troubleshooting complex scenarios
- When the core SKILL.md instructions need supplementary detail

## Patterns

### Pattern A: Direct Invocation
The user explicitly asks for this skill by name or describes a matching task.

### Pattern B: Contextual Activation
Claude recognizes that the current task matches this skill's description and loads it automatically.
